# What These Students Are Learning

All three courses are taught by Saied Moezzi at both CHS and SCHS.

## Intro to Computer Science (CHS Period 1, SCHS Period 4)

**Who they are:** First-time coders. No prior experience assumed.

**Language:** Python

**What they're learning:**
- Variables, data types, conditionals, loops, functions, basic data structures
- Building interactive programs and applied projects
- 3-4 weeks of hands-on STEM: building/programming robots, flying drones, earning a certification
- How the internet works
- Web design -- creating their own websites

**Course standing:** Standards-aligned, A-F course approval (meets college entrance requirements).

**What they care about:**
- "Can I make something cool?" (robots, drones, websites, interactive programs)
- Immediate visual feedback
- Not feeling lost or stupid
- Whether CS is "for them" at all
- Working with peers

**What they don't know:**
- Object-oriented programming
- Data structures beyond basics
- Sorting/searching algorithms
- Recursion, file I/O, anything formal

## PLTW Computer Science Principles (SCHS Period 5, mixed with AP CSA)

**Who they are:** College-level course. Project-based. Emphasizes creativity, collaboration, real-world problem solving.

**What they're learning:**
- Programming
- Data analysis
- Cybersecurity
- The internet (how it works)
- Societal impacts of computing
- Designing and developing apps
- Analyzing real-world data
- How computing innovations influence everyday life

**Course standing:** College-level. Students who get B or better receive 3 credits from MiraCosta College. This is the PLTW (Project Lead The Way) version, which is project-based and hands-on.

**What they care about:**
- Understanding how real-world tech works
- Building things (apps, data projects)
- Getting college credit
- Ethical and social dimensions of computing
- Computational thinking as a transferable skill

## AP Computer Science A / Advanced Computer Programming -- Java (SCHS Period 5, mixed with CSP)

**Who they are:** College-level. The most advanced students. Called "AP CS A" at CHS and "Advanced Computer Programming" at SCHS.

**Language:** Java

**What they're learning:**
- Object-oriented programming
- Problem-solving and algorithm development
- Data structures (arrays, ArrayLists)
- Code analysis
- Designing classes, methods, variables
- Control structures (loops, conditionals)
- Computational thinking and logical reasoning
- Writing and testing code

**2025-2026 curriculum (recently revised into 4 units):**
- Unit 1: Objects, Methods, Expressions
- Unit 2: Control Flow (booleans, conditionals, loops, efficiency analysis)
- Unit 3: Class Design (constructors, encapsulation, access modifiers)
- Unit 4: Data Collections (arrays, ArrayLists, 2D arrays, searching, sorting, recursion, file I/O)

**Notable change:** Inheritance/polymorphism removed from the 2025-2026 exam.

**Course standing:** AP exam may earn college credit. Students who get B or better also receive 3 credits from MiraCosta College.

**What they care about:**
- Mastering a professional language
- Getting college credit
- Building toward a CS major or tech career
- Solving algorithmic problems
- How software is actually designed and built

## The Mixed Period 5 Challenge

Period 5 at Sage Creek has PLTW CSP and AP CSA students in the same room. This means:
- Wide skill range: some students build apps and analyze data, others write Java classes and algorithms
- Both groups are college-level, so neither is a beginner -- the gap is depth, not starting point
- Need content that's accessible to CSP students but not boring for CSA students
- Connecting to shared themes (how software works in the real world, career paths, AI) bridges the gap
- Live demos and storytelling work for both levels; deep Java syntax discussions don't
- Both groups can earn MiraCosta College credit, so they're invested
