# What These Students Are Learning

## Intro to Computer Science (CHS Period 1, SCHS Period 4)

**Who they are:** First-time coders. No prior experience assumed. Likely 9th-10th graders.

**What they know:**
- Block-based coding (Scratch, Snap!, MakeCode) transitioning to text-based (Python or JavaScript)
- Variables, conditionals (if/else), loops (for/while), functions
- Event-driven programming (click handlers, key presses)
- Lists/arrays at a basic level
- Pair programming, debugging
- Likely working toward a capstone project (game, app, animation)

**What they don't know:**
- Object-oriented programming
- Data structures beyond basic lists
- Sorting/searching algorithms
- Recursion, file I/O, anything formal

**What they care about:**
- "Can I make something cool?" (games, animations, apps)
- Immediate visual feedback
- Not feeling lost or stupid
- Whether CS is "for them" at all
- Working with peers

## Computer Science Principles (SCHS Period 5, mixed with AP CSA)

**Who they are:** Likely 10th-11th graders. Some may have taken Intro first.

**The 5 Big Ideas they're studying:**
1. Creative Development (program design, collaboration, debugging)
2. Data (binary, compression, visualization, data analysis)
3. Algorithms and Programming (variables, conditionals, loops, lists, procedures, efficiency)
4. Computer Systems and Networks (Internet protocols, routing, DNS, fault tolerance)
5. Impact of Computing (bias, digital divide, ethics, privacy, encryption)

**Key difference:** CSP is deliberately broad. Significant time on non-programming topics (how the internet works, data analysis, societal impact, ethics). The exam uses pseudocode, not a specific language.

**What they care about:**
- Understanding how real-world tech works
- Building a portfolio piece (Create Performance Task)
- Getting AP credit without intense math prerequisites
- Ethical and social dimensions of computing

## Advanced Computer Programming / AP CS A -- Java (SCHS Period 5, mixed with CSP)

**Who they are:** Likely 11th-12th graders with prior coding experience. The most advanced students.

**2025-2026 curriculum (recently revised):**
- Unit 1: Objects, Methods, Expressions (Java classes, String/Math methods)
- Unit 2: Control Flow (booleans, conditionals, loops, string traversal, efficiency analysis)
- Unit 3: Class Design (constructors, encapsulation, access modifiers, static vs instance)
- Unit 4: Data Collections (arrays, ArrayLists, 2D arrays, searching, sorting, recursion, file I/O)

**Notable change:** Inheritance/polymorphism removed from the 2025-2026 exam.

**What they care about:**
- Mastering a professional language
- Getting college credit (CS1 equivalent)
- Building toward a CS major or tech career
- Solving algorithmic problems
- How software is actually designed and built

## The Mixed Period 5 Challenge

Period 5 at Sage Creek has AP CSP and AP CSA students in the same room. This means:
- Wide skill range: some students know pseudocode basics, others write Java classes
- Need content that's accessible to CSP students but not boring for CSA students
- Connecting to shared themes (how software works in the real world, career paths, AI) bridges the gap
- Live demos and storytelling work for both levels; deep Java syntax discussions don't
